# croton-cholera-analysis
This repository contains ward level data and statistical analysis for the study of cholera outbreaks in New York City in 1832 and 1849, with particular focus on population density, housing conditions, and sanitary infrastructure between 1830 and 1850.


## Data
ward_level_data.csv
Frozen ward level dataset used for all analysis

data_dictionary.csv
Definitions and descriptions of variables

sources_and_notes.csv
Archival sources, citations, and methodological decisions
